# Security Policy

## Supported Versions

We support last 2 minor versions released.

| Version | Supported          |
| ------- | ------------------ |
| 2.6.x   | :white_check_mark: |
| 2.5.x   | :white_check_mark: |
| < 2.5   | :x:                |

## Reporting a Vulnerability

Please report security issues via security@sourefabric.org.
